# vrew 리뷰 데이터 분석(진행중)

📌 프로젝트 소개

이 저장소는 Vrew 서비스의 사용자 리뷰를 기반으로
주요 불편 요소를 파악하고, 개선 아이디어를 정리한 작은 분석 프로젝트입니다.

복잡한 기획 제안보다,
리뷰 데이터에서 발견되는 반복 패턴을 간단히 확인하고
이용자 경험을 더 좋게 만들 수 있는 포인트를 살펴보는 것을 목표로 합니다.

📌 주요 내용

- 앱·스토어 리뷰 데이터 수집

- 키워드/패턴 분석

- 자주 언급되는 불편 요소 정리

- 간단한 개선 아이디어 제안

📌 산출물

- 리뷰 분석 코드

- 주요 인사이트 요약

- 간단한 기능/UX 개선 아이디어
